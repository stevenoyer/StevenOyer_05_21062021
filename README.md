#### Pré-requis
- NodeJS
- NPM

#### Installation
Rendez-vous dans le dossier backend et faites

```cmd
npm install
```

Aucun serveur n'est requis pour ouvrir le frontend, vous pourrez ouvrir la page grâce à "Live Server" de Visual Studio Code
